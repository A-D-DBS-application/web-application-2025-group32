# Desk Reservation System - Group 32

## Our project
A web application for managing desk reservations in an office environment. This system allows employees to reserve desks, administrators to manage the office layout, and provides analytics on desk usage.

### Installation

1. Clone the repository:
```bash
git clone https://github.com/your-username/web-application-2025-group32.git
cd web-application-2025-group32
```

2. Install dependencies:
```bash
pip install -r requirements.txt
```

3. Set up your environment variables in `app/config.py`

4. Run the application:
```bash
python run.py
```
